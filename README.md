# A simple snake clone game for learning python. 


## Make virtual environment: 
```
python -m venv .venv
```

## Activate virtual environment: 
```
.\.venv\Scripts\activate
```

## Install libraries
```
pip install pygame
```

## Run game
```
py main.my
```
